⚡ Electric Bill Calculator
A streamlined Python-based application designed to help users calculate their monthly electricity costs with ease. By comparing current and previous meter readings, the tool provides an instant breakdown of the net units consumed and the total estimated bill.

🚀 Features
Usage Tracking: Automatically calculates net consumption based on current and previous month's units.
Instant Estimates: Real-time billing calculation based on pre-defined unit rates.
User-Friendly Interface: Built with Streamlit for a clean, interactive web dashboard experience.
Data Accuracy: Minimizes manual calculation errors for monthly household budgeting.

🛠️ Technical Stack
Language: Python
Framework: Streamlit (for the UI)
Logic: Standard arithmetic for unit differentiation and slab-based pricing.
